# MEMORY

Typed, approved facts that persist between conversations. `seed-memory` proposes the first entries. Nothing is saved here without approval. Every PROJECT entry carries an expiry date.

## Types

- **USER**: durable facts about me.
- **FEEDBACK**: a correction I gave, with the why.
- **PROJECT**: a time-bound fact about current work (must have an expiry).
- **REFERENCE**: a pointer to an external resource.

## Entries

- [USER] [fact about me] — why: [reason] — saved: [date]
- [FEEDBACK] [correction] — why: [reason] — saved: [date]
- [REFERENCE] [resource and where it lives] — why: [reason] — saved: [date]
